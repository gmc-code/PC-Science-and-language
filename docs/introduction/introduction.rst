====================================================
PC-Science-and-language introduction
====================================================

PC-Science-and-language is a resource for teachers and students to learn how to write about science using the Systemic Functional Grammar (SFG) framework. It provides guidance on how to structure scientific writing, use appropriate language features, and create cohesive and coherent texts.

The resources include explanations of different paragraph structures, such as PEEL and PRO, as well as examples of how to apply these structures in writing about various science topics. The aim is to help students develop their scientific literacy and communication skills by understanding the linguistic choices that make scientific writing effective.
