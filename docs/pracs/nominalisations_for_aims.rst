==========================================================
Scientific language register shift
==========================================================

*Understanding how everyday wording is converted into scientific language is essential for correctly identifying variables and writing aims.*


Key linguistic features
----------------------------

1. Nominalisation (main one)
- how far it goes → distance travelled
- how fast it happens → rate of change

2. Technical precision
- removes ambiguity
- makes variables measurable

3. Register tightening
- fewer verbs
- more noun phrases
- more abstract scientific terms

----

----

Structural Pattern for Scientific Writing
------------------------------------------

Some common transformations:

1. **Investigation wording (casual)**
   *how fast / how much / how far / how long*

2. **Variable wording (scientific noun)**
   *rate / time taken / distance / temperature / etc.*

3. **Aim wording (formal)**

   *To investigate the effect of the*
   *[independent variable]*
   *on the*
   *[dependent variable]*
