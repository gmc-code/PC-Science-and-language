============================================
Personal Errors
============================================

*Personal errors are one-off mistakes made by the operator.*

**Key features**

* Produce isolated outliers
* Not representative of measurement uncertainty
* Excluded from uncertainty calculations

**Common personal errors**

.. list-table::
    :header-rows: 1
    :widths: 28 72

    * - Personal Error
      - Explanation
    * - Operator / technique error
      - Procedure not followed correctly or equipment misused.
    * - Recording error
      - Data written down or interpreted incorrectly.
    * - One-off misreading
      - Scale read incorrectly due to inattention.

**Response to personal errors**

* Discard the incorrect result
* Repeat the measurement correctly
* Improve checking and recording procedures
