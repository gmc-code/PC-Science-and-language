============================================
Personal Errors
============================================

*Personal errors are one-off mistakes made by the operator.*

**Key features**

* Produce isolated outliers
* Not representative of measurement uncertainty
* Excluded from uncertainty calculations

**Common personal errors**

.. list-table::
   :header-rows: 1
   :widths: 16 22 32 30

   * - Source
     - Error Type
     - Explanation
     - Fix / Reduce by…
   * - Observational / Procedural
     - Operator / technique error
     - Procedure not followed correctly or equipment misused — e.g.
       adding reagents in the wrong order or not allowing a thermometer
       to equilibrate before reading.
     - Discard the invalid result; repeat using the correct technique.
   * - Observational / Procedural
     - Recording error
     - Data written down or interpreted incorrectly — e.g. transposing
       digits, recording the wrong units, or copying a value into the
       wrong row of a table.
     - Re-read or recheck the data record; correct or repeat the
       measurement if the original value cannot be recovered.
   * - Observational / Procedural
     - One-off misreading
     - Scale read incorrectly due to inattention — e.g. reading the
       wrong scale on a dual-scale instrument or misreading a meniscus
       under poor lighting.
     - Discard the invalid result; re-read the instrument carefully
       and record the correct value.
