Types of Errors in Scientific Experiments
==========================================

*Understanding error types helps evaluate the accuracy, precision, and
validity of experimental results.*

This resource uses a **two-level model** consistent with VCAA, ACARA, and
major Australian secondary science textbooks (Pearson, Oxford, Nelson):

*  **Level 1 — Evaluation layer:** classify errors as random, systematic,
   or personal. This is what VCAA assesses.
*  **Level 2 — Explanation layer:** identify the source (instrumental,
   observational/procedural, method, or environmental). This is used to
   explain *why* an error occurred and *how* to fix it.

.. note::

   **⚠ VCAA Exam Note**

   The source categories below are a teaching framework. For VCE exams
   (Biology, Chemistry, Physics, Psychology), VCAA requires you to classify
   all errors as **random** or **systematic** — and to treat operator
   mistakes and recording errors as **personal errors**, which are *distinct*
   from true measurement errors.


How to Analyse Any Error: A Four-Step Framework
------------------------------------------------

Use these four steps whenever you are asked to identify, explain, or
evaluate an error in an investigation.

**Step 1 — Identify the source** *(what caused it?)*

Ask: where did the error come from?

*  **Instrument** — zero error, calibration fault, resolution limit
*  **Observation / procedure** — parallax, technique, recording mistake
*  **Method** — design flaw, built-in assumption
*  **Environment** — temperature fluctuation, vibration, lighting

**Step 2 — Classify the behaviour** *(how does it affect the data?)*

Ask: is the effect consistent or unpredictable?

*  Consistent, one-direction shift → **Systematic** → affects accuracy
*  Unpredictable spread → **Random** → affects precision
*  A one-off mistake → **Personal error** → discard and repeat

**Step 3 — Explain the impact** *(what does it do to your results?)*

*  **Systematic:** all results shifted too high or too low by a similar amount
*  **Random:** results scattered around the true value
*  **Personal error:** single outlier, not representative of the measurement

**Step 4 — Suggest an improvement** *(this is what VCAA marks)*

*  **Systematic** → eliminate the source: recalibrate, redesign, correct
   technique
*  **Random** → repeat measurements and average; increase sample size
*  **Personal error** → repeat the measurement correctly

Source Category 1: Instrumental Limitations
--------------------------------------------

*Errors arising from the measuring device itself.*

.. list-table::
   :header-rows: 1
   :widths: 22 50 28

   * - Error Type
     - Description
     - VCAA Classification
   * - **Zero Error**
     - Occurs when the instrument does not read zero when it should
       (e.g. a balance reading 0.2 g with nothing on it).
     - Systematic — consistently shifts all readings in one direction.
   * - **Calibration Error**
     - Occurs when the instrument is incorrectly calibrated or faulty,
       giving inaccurate readings across its range.
     - Systematic — affects accuracy; cannot be fixed by repeating
       measurements.
   * - **Resolution Limitation**
     - Precision is limited by the smallest scale division of the instrument
       (e.g. a ruler marked in mm cannot resolve 0.5 mm).
     - Random — introduces spread when estimating between graduations.

.. tip::

   Check instruments read zero before use and confirm they are within
   calibration. The resolution limitation is inherent — report measurements
   to the precision the instrument allows.


Source Category 2: Observational / Procedural Errors
------------------------------------------------------

*Errors arising from how measurements are taken or recorded.*

.. list-table::
   :header-rows: 1
   :widths: 22 50 28

   * - Error Type
     - Description
     - VCAA Classification
   * - **Parallax Error**
     - Occurs when a scale is read from an incorrect viewing angle, causing
       the apparent reading to differ from the true value.
     - Systematic — consistently over- or under-reads in the same direction.
   * - **Operator / Technique Error** ★
     - Occurs when equipment is used incorrectly or the correct procedure is
       not followed (e.g. not waiting for a thermometer to equilibrate).
     - Personal error — treat as a mistake; eliminate by repeating the
       measurement correctly.
   * - **Method Limitation**
     - A flaw in the experimental design itself (not the operator) that
       introduces bias — e.g. using a coloured solution in a titration
       without a suitable indicator.
     - Systematic — a source of systematic error; cannot be corrected by
       repetition.
   * - **Recording Error** ★
     - Occurs when data are written down or interpreted incorrectly
       (e.g. transposing digits, misreading units).
     - Personal error — treat as a mistake; correct by careful checking and
       re-reading.

★ Operator and recording errors are classified as **personal errors** by
VCAA — they are mistakes, not true measurement errors, and should not be
included in uncertainty calculations.


Source Category 3: Environmental Influences
--------------------------------------------

*Errors arising from uncontrolled conditions in the experimental setting.*

.. list-table::
   :header-rows: 1
   :widths: 22 50 28

   * - Error Type
     - Description
     - VCAA Classification
   * - **Temperature Variation**
     - Uncontrolled temperature changes affect readings (e.g. thermal
       expansion of apparatus, enzyme activity changes in Biology).
     - Random if fluctuating unpredictably; systematic if consistently
       higher or lower than the controlled condition.
   * - **Vibration / Interference**
     - Mechanical vibration or electromagnetic interference causes
       instruments to give unstable readings.
     - Random — introduces unpredictable spread in results.
   * - **Lighting Conditions**
     - Poor or inconsistent lighting affects how scales are read or how
       colour-based indicators are interpreted.
     - Systematic if consistently biased; random if varying between reads.

.. tip::

   Environmental influences are particularly relevant in Biology and
   Chemistry. Always control or monitor environmental conditions and note
   them in your method. In VCAA exams, naming a specific environmental
   variable (e.g. "room temperature fluctuations") is stronger than writing
   "environmental error."


Evaluation Layer: Random and Systematic Errors
-----------------------------------------------

*This is the primary classification VCAA uses to assess your understanding
of error. Every source error from Categories 1–3 maps onto one of these.*

.. list-table::
   :header-rows: 1
   :widths: 18 44 18 20

   * - Error Type
     - Description
     - Effect on Data
     - Reduced by…
   * - **Random Variation**
     - Unpredictable fluctuations between measurements. Sources include
       instrument resolution limits, environmental changes, and slight
       procedural variation.
     - Reduces **precision**
     - Repeating and averaging; larger sample size
   * - **Systematic Bias**
     - A consistent shift of all measurements in one direction from the
       true value. Sources include faulty calibration, zero error, parallax
       error, method flaws, and consistent environmental conditions.
     - Reduces **accuracy**
     - Identifying and eliminating the source; repeating does **NOT** help
   * - **Personal Error**
     - A one-off mistake by the operator — not a true measurement error.
       Examples: misreading a scale, transposing digits, using the wrong
       reagent.
     - Produces an **outlier**
     - Discarding the result; repeating the measurement correctly
