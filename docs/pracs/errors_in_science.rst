Types of Errors in Scientific Experiments
==========================================

*Understanding error types helps evaluate the accuracy, precision, and
validity of experimental results.*

This resource uses a **two-level model**:

* **Primary classification (assessment level):**
  **Systematic errors**, **Random errors**, and **Personal errors**
* **Secondary classification (explanation level):**
  Instrumental, observational/procedural, method, or environmental sources

----

How to Analyse Any Error: A Four-Step Framework
-----------------------------------------------

Use this framework whenever you are asked to identify, explain, or evaluate
sources of error.

**Step 1 — Identify the source** *(what caused it?)*

* **Instrument** — zero error, calibration fault, resolution limit
* **Observation / procedure** — parallax, technique, recording mistake
* **Method** — design flaw, built-in assumption
* **Environment** — temperature fluctuation, vibration, lighting

**Step 2 — Classify the behaviour** *(how does it affect the data?)*

* Consistent, one-direction shift → **Systematic error** → affects accuracy
* Unpredictable spread → **Random error** → affects precision
* One-off mistake → **Personal error** → discard and repeat

**Step 3 — Explain the impact**

* **Systematic:** all results shifted too high or too low
* **Random:** results scattered around the true value
* **Personal:** isolated outlier, not representative

**Step 4 — Suggest an improvement**

* **Systematic** → eliminate the source (recalibrate, redesign, correct setup)
* **Random** → repeat and average; increase sample size
* **Personal** → repeat the measurement correctly

----

1. Systematic Errors (Affect Accuracy)
--------------------------------------

*Systematic errors cause a consistent bias in all measurements.*

**Key features**

* Results are consistently too high or too low
* Repetition does **not** reduce the error
* Accuracy is reduced

**Common sources of systematic error**

.. list-table::
    :header-rows: 1
    :widths: 22 38 40

    * - Source
      - Error Type
      - Explanation
    * - Instrumental
      - Zero error
      - Instrument does not read zero, shifting all values.
    * - Instrumental
      - Calibration error
      - Scale or sensor is incorrectly calibrated.
    * - Observational
      - Parallax error
      - Scale consistently viewed from an incorrect angle.
    * - Method
      - Method limitation
      - Experimental design introduces inherent bias.
    * - Environmental
      - Constant temperature / lighting bias
      - Conditions consistently differ from intended controls.

**Reducing systematic errors**

* Recalibrate or replace equipment
* Redesign the method
* Correct reading position or setup
* Control environmental conditions

----

1. Random Errors (Affect Precision)
-----------------------------------

*Random errors cause unpredictable variation between measurements.*

**Key features**

* Results vary unpredictably around the true value
* Precision is reduced
* Averaging improves reliability

**Common sources of random error**

.. list-table::
      :header-rows: 1
      :widths: 22 38 40

    * - Source
      - Error Type
      - Explanation
    * - Instrumental
      - Resolution limitation
      - Estimation between scale divisions varies.
    * - Environmental
      - Temperature fluctuation
      - Small uncontrolled changes affect readings.
    * - Environmental
      - Vibration / interference
      - Readings fluctuate due to instability.
    * - Procedural
      - Minor technique variation
      - Small unavoidable variations between trials.

**Reducing random errors**
* Repeat measurements and calculate a mean
* Increase sample size
* Use higher-resolution instruments
* Stabilise environmental conditions

----

1. Personal Errors (Mistakes, Not Measurement Errors)
-----------------------------------------------------

*Personal errors are one-off mistakes made by the operator.*

**Key features**
* Produce isolated outliers
* Not representative of measurement uncertainty
* Excluded from uncertainty calculations

**Common personal errors**

.. list-table::
    :header-rows: 1
    :widths: 28 72

    * - Personal Error
      - Explanation
    * - Operator / technique error
      - Procedure not followed correctly or equipment misused.
    * - Recording error
      - Data written down or interpreted incorrectly.
    * - One-off misreading
      - Scale read incorrectly due to inattention.

**Response to personal errors**
* Discard the incorrect result
* Repeat the measurement correctly
* Improve checking and recording procedures
